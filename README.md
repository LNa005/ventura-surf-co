**Ventura Surf Co.** es una landing page de alto rendimiento diseñada para una escuela de deportes acuáticos en Fuerteventura. El proyecto combina una estética editorial moderna con optimizaciones técnicas de nivel profesional.

---

## 🚀 Características Principales

### 💎 Diseño y Experiencia de Usuario (UX)
* **Cursor Personalizado Suave:** Implementado mediante `requestAnimationFrame` para una navegación fluida y orgánica.
* **Diseño Editorial:** Tipografías contrastadas (`Bebas Neue`, `Syne`, `Crimson Pro`) y paleta de colores inspirada en el entorno volcánico de Canarias.
* **Totalmente Responsivo:** Adaptado meticulosamente para dispositivos móviles y tablets.
* **Interacciones Dinámicas:** Gráficos de viento animados y tickers de texto infinito.

### 🛠️ Especificaciones Técnicas (Performance & SEO)
* **Optimización de Animaciones:** Uso de un bucle de animación con **Interpolación Lineal (Lerp)** para reducir el uso de CPU en el seguimiento del cursor.
* **SEO & Open Graph:** Configuración completa de Meta Tags para una indexación óptima y previsualizaciones atractivas en redes sociales.
* **Datos Estructurados (JSON-LD):** Implementación de esquema `Course` e `ItemList` para habilitar *Rich Snippets* en los resultados de búsqueda de Google.
* **Accesibilidad (A11y):** Uso de clases `.sr-only` para lectores de pantalla y etiquetas semánticas vinculadas correctamente.

### 📩 Funcionalidad de Negocio
* **Formulario de Reserva Real:** Integración con **Formspree** para la captura de leads directamente al correo electrónico, incluyendo validaciones nativas de HTML5.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5** (Semántica avanzada)
* **CSS3** (Custom Properties, Flexbox, Grid, Animaciones avanzadas)
* **JavaScript ES6+** (Intersection Observer API, rAF Animation Loop)
* **JSON-LD** (SEO Semántico)
